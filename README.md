## ツール

Python3
Pytorch
torchvision
Django

## 必要ライブラリのインストール
ターミナル上で以下のコマンドを実行てください。

pip install django torch torchvision opencv-python


### アプリの起動と動作確認

以下のコマンドでwebアプリを起動

python manage.py runserver

### 動作確認

[http://127.0.0.1:8000 ](http://127.0.0.1:8000/flower/upload)にアクセスできるか確認
アクセスできたら成功
